# dotfiles

My current setup on ArchLinux:

- **Login**: GDM
- **WM**: Niri (Wayland, scrollable-tiling)
- **Desktop shell**: Noctalia (quickshell)
- **Terminal**: Ghostty
- **Shell**: zsh + zinit
- **Prompt**: oh-my-posh
- **Editor**: Neovim
- **Multiplexer**: tmux + tms
- **Audio**: PipeWire + WirePlumber
- **Theme**: Catppuccin Mocha
- **Browser**: Brave
